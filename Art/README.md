# Art — рабочие арты CHIMERA

Сюда скидываем **концепты, референсы, скетчи, наброски** — всё «сырое» от арт-команды.

Почему тут, а не в `Assets/`: Unity импортирует всё внутри `Assets/` (плодит `.meta`, замедляет
импорт, тащит в билд-обработку). Рабочие материалы живут здесь; в `Assets/_Chimera/` переезжает
только готовое к использованию в игре (модели/текстуры/спрайты).

Подпапки — по смыслу, договоримся по ходу; для старта:

- `concepts/` — концепты существ, тел, органов, локаций
- `refs/` — референсы (чужие картинки, мудборды)
- `ui/` — наброски интерфейса (конструктор-«тетрадь да Винчи», созвездия)

Стартовые ориентиры арт-брифа (из GDD, §визуал; это трамплин, не канон): high-performance low-poly,
«сочность» через свет/туман/пост, химеризация выглядит органично (тело отращивает черты, не хирургия),
эталон вервольфа — вервольф из «Ван Хельсинга», конструктор — анатомическая тетрадь да Винчи,
родство — звериные созвездия (Lupus, Serpens…).

⚠️ Большие файлы: git без LFS — гигабайтные PSD лучше не коммитить как есть (экспортируйте
PNG/JPG разумного размера; про LFS договоримся, когда припрёт).
