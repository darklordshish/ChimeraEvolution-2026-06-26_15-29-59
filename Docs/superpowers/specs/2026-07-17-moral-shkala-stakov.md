# МОРАЛЬ — динамическая шкала стаков (страх ↔ ярость) + тень-загон лося

**Статус:** дизайн пользователя 2026-07-17, на ревью. Носитель №1 — волк (стайный); лось/вервольф не мигрируют (их ярость — берсерк/самоподдержка, не мораль).

## 1. Проблема

- `Fear` копится величиной с **невидимым затуханием** (0.5/с — магическая ручка); `Rage` от воя — **немедленный** бафф урона. Ни решение «нападать на опасную добычу», ни перетягивание духа сценой не выразимы.
- Для охоты стаи на лося нужен **перелом морали**: теллы лося давят, вои качают, у кого перевесит.

## 2. Модель: вклады с временем жизни

Компонент **`Morale`** (стайные психики; холоднокровным/одиночкам не вешается):
- **ЕДИНАЯ АРИФМЕТИКА (упрощение пользователя): любое событие = ±1 на 10 секунд** — различие только ЗНАКОМ. Мораль = сумма живых вкладов, кламп ±5. Затухания нет — вклады просто истекают. «Завыл = +1; через 2с ещё вой — на 8-й секунде их два; смерть −1 — и вот +1; ещё смерть — 0, и побежали».
- Единственное исключение: **вой ВОЖАКА (вервольф) = +5** — «приказ в атаку», мгновенно перекрывает любой порог храбрости (замена Rally). КД его воя 16с при жизни стака 10с — приказ действует окнами, не вечным фоном.

**Источники: вой сородича +1 · вой вожака +5 · смерть сородича рядом −1 · РЁВ лося −1 · фырк/топот-теллы лося −1 · удар из невидимости −1. Всё ×10с (единый инкремент устоял после качелей: −2 за смерть ронял стаю постоянно). Пороги Bravery 2..4; финальный баланс плейтеста 2026-07-18: пороги 2..4 + всё ±1 + вой-не-хор (один голос/7с) + вожак-перекрытие (КД 9с < жизни стака).**
**Вой ИГРОКА — телесный вес (решение 2026-07-17): база −2 × бонус органов (родство ×1..×2) → до −4 на сотке с мощными аугументами — почти приказ вожака, только со знаком минус. Сила голоса растёт с волчностью тела.**

*Числа сходятся:* раскачать труса (порог 5) = пять ЖИВЫХ воёв разом — физически требует собранной стаи (вой каждого волка с КД ~5с держит ~2 живых стака) → «созыв до навала» вытекает из арифметики сам.

**Пороги — от личности (`Bravery` **1..3** — урок плейтеста M1: порог обязан жить в ЕДИНИЦАХ ВКЛАДОВ, «сколько смертей пугает»; старый 2..5 при ±1-инкрементах и 10с-окне был недостижим — стая не ломалась вообще; заодно вой — событие СТАИ, не хор: один голос в ~7с, иначе фон = размер стаи):**
- мораль ≤ −Bravery → **ПАНИКА**: rout. БЕЗ таймера — бежит, ПОКА ЖИВ СТРАХ (стаки истекут → вернулся; вой сородича может вытащить из паники досрочно). Ручка routDuration умирает вслед за decay.
- мораль ≥ +Bravery → **КОММИТ**: Rage-бафф (пока мораль выше порога) + разрешение атаковать опасную добычу. Трус позже и пугается-раскачивается — одна ось.
- **ГИСТЕРЕЗИС** (у порога мораль скачет ±1): вошёл в состояние — выходит на 1 НИЖЕ порога входа (коммит слетает при <порог−1, паника отпускает при >−порог+1) — без дёрганья «атака/бегство» каждые 2с.

**Осознанные изменения баланса (следить в плейтесте M1):** (а) живая воющая стая может стать почти несокрушимой (вои перекрывают редкие смерти) — если паника исчезнет как явление, крутить радиус/КД воя, НЕ веса; (б) испуг-вой игрока с мгновенного срыва (величина 6) падает до −1 — пугает только надломленных, инструмент становится «добиванием морали».

**Бонусы шкалы (плавные, не только пороги):**
- положительная → **бонус урона** (лин. до +25% на +5);
- отрицательная → **бонус скорости бегства** (до +30% на −5) — «страх окрыляет»; паникёр добегает до змеи быстро.

## 3. Судьба Fear/Rage (статусы остаются исполнителями)

- **Волк:** `Fear`-компонент замещается `Morale` (все его источники становятся −вкладами; `IsRouting` → мораль ниже порога). `Rage`-бафф теперь включается ТОЛЬКО коммитом (вой сам по себе урон не баффает — осознанное изменение баланса: стая слабее на старте, страшнее в раскачке).
- **Вервольф:** `permanent`-флаг Rage умирает — психика **сама поддерживает** свой стак (вечность = самоподдержка, как берсерк лося). Его вой = +2 стае.
- **Лось/змея:** не трогаются (берсерк/холодный расчёт). `EmotionTint`: паника волка — по морали.
- `PackCoordinator.Rally/Fearless` — вливается в шкалу (вой вожака = большой +вклад всем; отдельный флаг бесстрашия уходит).

## 4. Тень-загон лося (D-2)

- **Триггер:** волки ≥2 при виде лося (одиночка — держит дистанцию молча).
- **ТЕНЬ:** кружат на ~12м — сразу за границей лесенки лося (она от них ТИКАЕТ: лось фырчит/топает → −мораль волкам; его теллы = его оружие) — и **воют**, сзывая сородичей (`Hear` готов).
- Мораль пересекла коммит → **навал** (жетоны/battle-circle как на игрока; Bleed стекается — туша истекает). Лось ревёт/убивает → −вклады → паника → **рассып поодиночке** (корм змеям).
- **Лесенка лося** реагирует на «ближайшую видимую угрозу» (волки и игрок), не хардкод-игрока → «игрок стравливает лося со стаей» бесплатно.

## 5. Нарезка

- **M1**: `Morale` + миграция волка с Fear + вой→вклады (играбельно: прежние паники, но стаковые).
- **M2**: бонусы урона/скорости бегства + коммит→Rage + вервольф-самоподдержка.
- **M3**: тень-загон + охота на лося + лось-цель-угроза + теллы→−мораль.
- Полировка чисел — плейтестом (таблица выше — стартовая).
