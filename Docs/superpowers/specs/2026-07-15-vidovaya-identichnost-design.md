# CHIMERA — Видовая идентичность (kinship): признание вида по составу тела

- **Дата:** 2026-07-15
- **Статус:** дизайн на ревью пользователя → план по слайсам → код.
- **Зависит от:** сенсорного S1 (`Fear`, `Rage`, `Stagger`-стан, `ColdBlooded`), лосиной спеки (носитель — рёв), волчьей ветки (носитель — вой).
- **Отношение к спеке лося:** это ОТДЕЛЬНЫЙ арк (ядро S4). Его слот в конвейере — у среза **D** лосиной спеки (зависимость рёва); лосиное ядро (A–C) едет раньше. В лосиной спеке рёв пока на простом видо-теге — эта спека его ОБОБЩАЕТ.

## 0. Зачем

Механика того, **как состав твоего тела определяет, кем тебя считают обитатели** — своим или чужим. Резолв давнего вопроса GDD §12 «цена химеризации»: чем ты химернее, тем более чужероден миру. Раньше не строили — не было носителей; теперь их два (лосиный рёв + волчий вой), правило трёх позволяет вытащить общую механику, а не хардкодить «свой=вид» в каждом.

**Ключевой принцип:** признание считывается с ТЕЛА (шасси + надетые органы), НЕ с истории килов. Родство (килы→скидка) — отдельная ось (§5).

## 1. Метрика — «видовая идентичность %»

Для каждого вида X тело даёт число:

> **Идентичность%(X) = вес_шасси (если шасси = X) + Σ вес_органа (надетые родные органы X)**,
> нормировано так, что «шасси-X + все органы X = 100%».

Родство в формуле **не участвует**. Три градации-порога **признания вида**:

| Градация | Идентичность% | Смысл |
|---|---|---|
| **Слабое** | ≥ **65%** | порог БЕЗОПАСНОСТИ: «в основном свой» — не трогают, rally вялый (люфт на 1-2 примеси; мимикрия-стратегия) |
| **Среднее** | ≥ **85%** | порог ВЛАСТИ: безупречный облик (чистый моно-сет) — rally уверенный |
| **Сильное** | 100% | полностью свой (только родное шасси) — rally максимальный |

Пороги 65/85 — резолв пользователя 2026-07-18 (были 75/90 «впритык» к каноничным билдам — признание было лезвием без люфта). Ручки в инспекторе тела.

## 2. Шасси-как-кэп (= встроенная цена химеризации)

**Вес шасси ≈ 10%** (= разрыв между средним и сильным). Тогда:

- **Чужое шасси (человек) + ВСЕ органы X = 90% = СРЕДНЕЕ** — это ПОТОЛОК для чужого шасси. Игрок-человек, вложивший все слоты в один вид, максимум «приняли», но не «полностью свой».
- **Родное шасси + все органы X = 100% = СИЛЬНОЕ.** Полное признание требует **смены шасси** (post-MVP).

Так «цена химеризации» выпадает из ОДНОГО веса, без штрафа-костыля: человек всегда чуть чужероден, максимума родственной власти ему не взять. Вес шасси — ручка: больше вес → человек капится ещё ниже (вплоть до «без смены шасси признания нет»).

## 3. Вокализ: контроль-всех + rally-своих

Социальный вокализ (волчий **вой**, лосиный **рёв**) = ДОСТАВКА. Развязка эффекта на две части:

- **Контроль-эффект (от ОРГАНА-доставки)** ложится на ВСЕ цели в радиусе: вой → `Stagger`-стан; рёв → `Fear`-страх.
- **Rally-эффект (кин-цель от СОСТАВА тела)**: цели, к чьему виду ты кин (идентичность ≥ порога), ВМЕСТО контроля получают **призыв-в-союзники**; сила ∝ градации (радиус/интенсивность/стаки).

**Доставка ≠ кин-цель** — отсюда кросс-опыление (примеры пользователя):

- **Волчье тело + лосиный рёв** → остальным страх, а ВОЛКИ (кин) вместо страха идут в союзники.
- **Лосиное тело + волчий вой** → остальным стан, а ЛОСИ (кин) собираются на союзную атаку.

Т.е. «rally своих» не хардкодит вид — целит `MostKinSpecies(caster)`.

**Капстоун — «химера бьёт по всем»:** если кин ни к кому (родства всего по чуть-чуть) → rally никого → контроль ложится на ВСЕХ (рёв пугает всех, вой станит всех). Отсюда стратег-ось:
- **Специалист** (высокая идентичность к одному виду) = союзники этого вида + контроль остальных.
- **Химера** (размазан, кин ни к кому) = союзников нет, зато универсальный контроль на всех = «монстр для всех».

**Игрок-как-союзник (мясная часть):** если игрок кин к виду, его члены НЕ считают игрока угрозой — правка таргетинга психик (волки-кин не атакуют игрока, идут с ним, сбегаются по его вою). Это самое существенное изменение ИИ в арке (носитель «волчий вой»).

## 4. Змея — НЕ спец-исключение, а эмерджентное наложение

Уточнение (важно для понимания): идентичность%(Змея) считается **нормально**, как у любого вида — змее-кин по составу возможен, механика идентичности на змее не «выключена». Просто **rally и страх — это морале-эффекты** (`Rage`/`Fear`), а змея холоднокровна (`ColdBlooded`) и **вне системы морали** → эти эффекты по ней no-op. Отсюда следствие: снейк-идентичность не даёт вокального командования змеями — **не потому что идентичность отключена на змее, а потому что две независимые механики так наложились** (rally опирается на мораль, у змеи морали нет). Ничего специально не исключаем: `ColdBlooded` УЖЕ иммунен к `Rage`/`Fear`, эффекту просто не за что зацепиться. *(Физический стан от воя — не мораль, по змее в принципе может проходить; страх рёва — нет.)* Змея остаётся нейтралом-одиночкой без внутривидового соц-слоя как побочный итог, а не как авторское правило.

## 5. Отношение к родству (РАЗДЕЛЬНЫ, но связаны причинно)

- **Родство** = история килов → скидка на органы + бонус мощи. Как есть, **не трогаем**.
- **Видовая идентичность** = текущий состав тела. Новая ось.
- **Связь только upstream (причинная):** килы → скидка → дешевле органы вида → можешь набрать их больше → выше идентичность. Но в ФОРМУЛУ идентичности родство НЕ входит. Можно перебить сто волков (высокое родство) и ходить голым человеком (нулевая идентичность) — волки не признают. Разделение намеренное: родство — экономика, идентичность — удостоверение.

## 6. Числа и веса (ручки баланса)

- **Вес органа — РЕЗОЛВ (пользователь, 2026-07-18): ПЕРЕНОРМИРОВКА ПО МАССЕ ТЕЛА.** Части весят в
  «юнитах» СВОЕГО вида: орган = 0.9/N (N — аугумент-пул вида, РАВНЫЕ доли; chassisOnly — плоть шасси,
  живёт в его 0.1), шасси = 0.1. **Идентичность(X) = юниты X / масса всех надетых частей.** Свойства из
  построения: «все аугументы вида = 90%» у любого (змея 5×18%, волк 6×15%); Σ идентичностей = 100%;
  **примесь в химерном слоте РАЗБАВЛЯЕТ** (моно-волк 90% среднее → +пит-орган → 76% слабое — признание
  стаи падает от химеризации сверх сета). Ни перевзвешивания руками, ни отдельных шкал.
- **Вес шасси ≈ 10%** (чужое шасси → максимум среднее).
- **Пороги 75/90/100** (слабое/среднее/сильное) + маппинг градация → сила rally (радиус/интенсивность/стаки) — тюнинг.
- Придаток (хвост/рога) в химерном слоте — считается в идентичность своего вида (разбавляет «чистоту» другого); детали распределения — та же ручка.

## 7. Границы / YAGNI

НЕ сейчас: **скользящая непрерывная чуждость** (чем химернее — тем агрессивнее ВСЕ обитатели пассивно, не только на вокализ) — нет носителя, отдельный кусок S4; **полная кросс-видовая матрица реакций**; **притяжение других химер**; **смена шасси** (даёт сильное признание) — post-MVP; **точное распределение вкладов органов** — ручка баланса, дефолт «по цене». Player-fear «теряешь управление» (рёв ПО игроку) — бэклог GDD, не здесь.

## 8. Build-порядок / слайсинг

- **K1. Метрика идентичности.** `CreatureBody.Identity(species) → 0..1` + `KinTier(species) → нет/слабое/среднее/сильное` (чистое чтение состава: шасси-вес + сумма органо-весов). Дебаг-строка в HUD/дев-панели. Тестируется на игроке в конструкторе (надел/снял органы — число ходит; человек-моно кап на среднем).
- **K2. Носитель 1 — лосиный рёв.** Обобщает §9 лосиной спеки с видо-тега на идентичность: контроль-всех (`Fear`) + rally-кин (лоси → берсерк-каскад). Проверяет доставку≠кин-цель на реальном приёме.
- **K3. Носитель 2 — волчий вой-союзник.** Кин-игрок → волки не атакуют, идут с ним; контроль-остальных = `Stagger`-стан; игрок-моно-волк рулит стаей. Самая мясная правка (таргетинг `WolfPsyche`).
- **K4. Полировка/баланс по плейтесту** (веса, пороги, сила rally).

Слот в общем конвейере — у среза **D** лосиной спеки. Порядок K1→K2→K3 по зависимости (метрика раньше носителей; рёв проще воя-союзника, вой-союзник — самый техрисковый, поэтому последним).

## 9. Проверка (игрой)

- **Моно-волк-человек** (все слоты волчьи, человечье шасси): вой зовёт волков в союзники (среднее), станит остальных. Снял органы → падает до слабого/нет.
- **Химера** (микс видов): вой станит ВСЕХ, союзников нет («монстр для всех»).
- **Змея**: снейк-кин НЕ командует змеями — не спец-правило, а наложение (rally/страх морале-based, змея холоднокровна вне морали); идентичность%(Змея) при этом считается нормально.
- **Кросс-опыление**: лосиное тело + волчий вой → сзывает ЛОСЕЙ, не волков.
- **Родство ≠ идентичность**: перебил стаю волков (родство ↑), но голый — волки не признают.

## Термины

- **Видовая идентичность** — метрика по составу (шасси + органы), 0–100%.
- **Признание вида** — порог идентичности: слабое (75) / среднее (90) / сильное (100).
- **Родство** — существующая ось (килы → скидка/мощь), ОТДЕЛЬНА от идентичности.
- **kinship** — англо-сокращение всей механики (для кода: `Identity`/`KinTier`).
