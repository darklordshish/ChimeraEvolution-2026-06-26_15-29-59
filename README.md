# CHIMERA

Рогалик «по флоу» про последнего безумного учёного, который химеризует собственное тело органами убитых зверей — чтобы выжить среди сбежавших химер и добраться до сердца окуклившейся лаборатории.

**Сюжетная рамка.** Лаборатория создавала Химеру 0 — идеальную адаптивную химеру — на оригинальной мутагенной сыворотке. Вспышка: Химера 0 заразила коллег (они стали боссами-суперхимерами — трагическими жертвами, не злодеями), зверинец разбежался и одичал. Герой вкалывает себе чистую сыворотку по своей воле: управляемая химеризация — его оружие. Химера 0 тем временем пожирает и мутирует — растущая угроза и финальный босс.

**Ядро геймплея.** Убиваешь зверей → копится **родство** с видом → звериные органы дешевеют → в слот-конструкторе размениваешь человеческие органы на звериные (пул мутагена = ёмкость) → тело реально меняется: способности, чувства, статы. Вариативность «по флоу, а не по луту»: билд вырастает из того, каких зверей подкинул маршрут забега.

## Текущее состояние (прототип)

Играется сейчас:

- **Три вида врагов**: стая волков (окружение, жетоны атаки, захват, мораль/бегство, вой), змеи-засадницы (термозрение, яд, 3-стадийный удушающий обхват, невидимость-в-неподвижности + гремок), босс **Вервольф** — первая суперхимера (чардж, вой-призыв, вампиризм; первое убийство даёт +пул и химерный слот).
- **Конструктор химеры** (Tab, слоу-мо ×10): 6 слотов + химерный слот, мультидонор (человек ↔ волк ↔ змея на слот), экономика пула со скидками от родства, супремум-агрегация дублей.
- **Экосистема чувств (камень-ножницы-бумага)**: зрение (рвётся стенами) ↔ нюх (запаховые тропы) ↔ термозрение (тепло сквозь стены, слепо к холоднокровным) ↔ скрытность (камуфляж, гремок — единственная зацепка).
- **Способности от органов**: укус (вампиризм ↔ яд), пинок, вой-стан, рывок, обхват хвостом (контроль-удержание).
- Процедурная арена-лабиринт с NavMesh, спавнеры, отладочный HUD и Dev-панель.

## Управление

ЛКМ меч · ПКМ/Space рывок · E пинок · Shift укус · Alt вой · F обхват · Tab конструктор · V вид 1/3 лицо · 1–7 органы.
Отладка: G бог · K/L +10 родства (волк/змея) · N свой запах · T форс-термо · `Chimera → Dev-панель` в редакторе.

## Как открыть

1. Unity **6000.4.7f1** (URP, новый Input System, AI Navigation 2.0).
2. Открыть папку проекта → сцена `Assets/Scenes/SampleScene.unity` → Play.
3. Данные видов уже в репо (`Assets/_Chimera/Data/`); после правок `SpeciesBootstrap` — меню **Chimera → Создать дефолтные виды**. Префабы существ пересобираются генераторами (**Chimera → Создать префаб …**), не руками.

## Архитектура (кратко)

- **Тело = данные**: `CreatureBody` + `SpeciesSO` (шасси + доноры + экспрессия) — общее для игрока и NPC.
- **Психика = тонкий код**: решения вида (sense → think → act) поверх общих «глаголов тела».
- **Способность = доставка + эффекты**: переиспользуемые доставки (`BiteAbility`, `LeapAbility`…) × словарь эффектов (`Hit`/`HitEffect`).
- Статусы — компонентами (`Rage`, `Venom`, `Stagger`, `ColdBlooded`, `Camouflage`).

Подробно — в [Docs/CONSTRUCTOR_GUIDE.md](Docs/CONSTRUCTOR_GUIDE.md) (читать перед любой работой с видами).
